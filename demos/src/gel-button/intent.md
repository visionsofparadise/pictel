# Gel Button

A glossy candy-blue web button floating above a dark indigo backdrop — a radial body shading from pale sky-blue at the top to deep navy at the bottom, an oval top-of-button highlight giving the bubble-of-light effect characteristic of Aqua-era UI buttons, the word GO set in bold sans across the centre, and a soft offset shadow giving the button lift off the page. The look is the classic 2003 OS X dock-button aesthetic, deliberately glassy and over-illuminated.
